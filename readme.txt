2. Неавторизованный пользователь.
	Имеет доступ к главной странице сайта. Видит список всех машин с ограниченным набором полей. Может осуществить поиск
	по номеру машины (полное соответствие). Нажав на строку таблицы видит подробности о машине (органиченное число полей).
	На другие страницы попасть не может, даже по прямой ссылке.
	По API может получить список всех машин (с ограниченным числом полей) и инофрмацию о конкретной машине (с ограниченным числом полей).
	Пользовтаель не может самостоятельно зарегистрироваться в системе. По ссылке http://127.0.0.1:8000/accounts/signup/ будет написано, что регистрация закрыта. Завести нового пользователя может только admin через админ. панель.
	
	Клик по логотипу в левом верхнем углу ведет на главную страницу.

3. Клиент.
	На стартовой странице функционал тот же, что и у неавторизованного пользователя.
	Доступны кнопки: "Мои машины", "ТО", "Рекламации", "Добавить ТО"
	
	"Мои машины": список машин клиента, отсортированных по дате отгрузки с завода (согласно ТЗ проекта). Есть возможность фильтрации по 5 полям. По ссылке на строку таблицы открывается подробная информация о машине.
	
	"ТО": Список ТО, созданных клиентом, отсортированный по дате проведения ТО. Есть возможность фильтрации по 3 полям. В поле "заводской номер машины" отображается список только из машин клиента.
	
	По клику на строке таблицы открывается подробная информация о ТО. Внизу присутсвуют кнопки "изменить" и "удалить". Попасть на чужие ТО клиент не может даже по прямой ссылке. Так же как на страницы их изменения и удаления.
	
	"Рекламации": Список рекламаций, оставленных сервисной коспанией на машины клиента, отсортированный по дате отказа. Есть возможность фильтрации по 3 полям. По клику на строке таблицы открывается подробная информация о рекламации. Попасть на рекламации к чужим машинам клиент не может даже по прямой ссылке.
	
	"Добавить ТО": Откроется форма для создания нового ТО на одну из машин клиента. Её нужно заполнить и нажать "Сохранить".
	
	
4. Сервисная компания.
	Те же фукции, что и у клиента за исключением:
	- на главной странице появляется кнопка "доб. рекламацию" для добавления рекламации от сервисной компании.
	- в "моих машинах" теперь список машин, приписанных этой сервисной компании.
	- в "ТО" так же список машин, приписанных этой сервисной компании. В фильтре "заводской номер машины" тоже, только машины, приписанные этой сервисной компании.
	- в "рекламациях" - рекламации написанные этой сервисной компанией. В рекламациях появлеются кнопки "изменить" и "удалить".
	- все остнальные ТО и рекламации по прямым ссылкам недоступны.

5. Менеджер.
	- Полные права на создание/изменение/удаление любой машины, ТО или рекламации.
	- на главной странице появляется кнопка "доб. машину" для добавления новой проданной машины.


6. Общая информация.
	- на страницах с таблицами используется пагинация по 10 записей.
	- сделана резиновая верстка вплоть до ширины в 360 пикселей. (на маленьких разрешениях уменьшается число столбцов в таблицах, для читаемости). 
	- использованы фирменные цвета, логотипы, шрифты и иконки фирмы Силант.
	- дизайн основан на предоставленном макете.
	- сайт сделан на Django + шаблонизатор
	- приложение service - сам сайт, приложение api - REST API к сайту
